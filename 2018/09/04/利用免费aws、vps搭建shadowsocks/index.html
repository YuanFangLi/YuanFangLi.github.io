利用免费aws vps搭建shadowsocks

1：https://ap-southeast-1.console.aws.amazon.com 在此站注册账号
自己已经有了账号了，没法发图没法写了，google吧，首选你得有个信用卡
2：新建免费vps
都有中文了，最好是选小日本的或者新加坡的，不过貌似新建出来的IP都是美国的，这个我就搞不明白了，也许是因为免费的吧，点击启动实例图片
3：选择ubuntu，记得吧仅免费套餐勾选上，要不扣钱是不通知你的，扣完才有账单
图片
4:选择免费套餐，下一步
图片
5:启用终止保护最好勾选上，租赁选择共享，专用是收费的。下一步
图片
6: 默认，8g够了，不能小于8g，大于30g收费，下一步
图片
7: 不截图了，直接下一步
8: 选择一个现有的安全组，用默认的所有流量就好，不然还得自己进去开端口，麻烦。
图片
9: 点击审核和启动后，直接点击启动。
10:选择创建新密钥对，输入名称，当然，不能是中文，然后下载，下载后点击启动实例
图片
11: 点击此货就可以进免费对vps了
图片
12:进去之后首选修改安全组的入站规则，点击安全组， 点击“操作” 编辑入站规则
图片
13:选择我的IP，保存，vps就OK了，下面就是连接vps安装shadowsocks服务了

14:使用ssh客户端连接vps，官方就有教程，win系统可以用PuTTY连接，Mac Linux直接终端好的很，Mac Linux下需要修改下载的密钥权限，官方也有提示，win下的应该不要，不清楚
https://docs.aws.amazon.com/zh_cn/AWSEC2/latest/UserGuide/putty.html 官方 PuTTY教程

15:连接好之后就可以安装shadowsocks服务了，安装服务
1:在终端输入 ：sudo su 获取root权限
2:apt-get update 更新软件源列表
3:apt-get install python-gevent python-pip 安装所需服务
4:pip install shadowsocks 安装shadowsocks
5:vi /etc/shadowsocks.json 新建一个 config.json，（可以是其他路径，随便）按i键进入编辑模式 终端下方应出现”insert”字样 将下面的文件拷贝到文本文档 并进行修改

{
“server”:”0.0.0.0”,
“server_port”:0000,
“local_address”: “127.0.0.1”,
“local_port”:1080,
“password”:”password”,
“timeout”:300,
“method”:”table”,
“fast_open”: false,
“workers”: 1
}
你需要修改的:
“server”:服务器ip地址(从VPS提供商那里取得，aws的IP地址为0.0.0.0，不然会出错) “server_port”:服务器端口号:8388 “password”:密码 “method”:加密方法 最好是aes-256-cfb
一切修改好并复制到文件后，按ESC，输入“:wq”(不含引号)按回车，退出vi
为了支持这些加密方式，你要需要安装

apt-get install python-m2crypto

然后就可以启动服务了。

6.运行 ssserver -c /etc/shadowsocks.json (或者上文中你所设置之路径)
如果这一步没有提示出错，就可以配置自动开机了
然后可以配置开机自启动

vi /etc/rc.local（把下面一行添加进去，然后esc :wq）

/usr/local/bin/ssserver -c /etc/shadowsocks/config.json

重启服务器，reboot，OK了
